To work on denoting the duplicate records all the database data was imported into excel and a new column was created. The column called 'dupe_record' was then filled with a 0 or 1 indicated either that this was the first time a record had been seen or if the record was a duplicate. Along with the general schema of the ID number being a uniuqe number and this new flag cloumun the import went smoothly.

This was done via a nice website called https://sqlizer.io/ that converts excel tables into sql commands.

Commands:

alter table proj2_database
add column contact_pref VARCHAR(36) CHARACTER SET utf8 DEFAULT 'DNC' after Phone_number;

alter table proj2_database
add column email_addr VARCHAR(255) CHARACTER SET utf8 after Phone_number;

Notes: 

Created two columns one per project spec and another to hold email address's if needed.
The reason the contact_pref is defaulted to 'DNC' is to reduce checks seeing as it will probably be updated. And all contact should be an opt-in not opt-out.

The commands below to update the table are taliored to there specific needs and some do not go through.
In this case only the earlist record is the one that is used for contact information.

# John Wick
update proj2_database
set contact_pref = 'ROBOCALL'
where id = 2 AND (Phone_number IS NOT NULL AND dupe_record = 0);

# Tony Stark slight issue no email but will get robocalls
update proj2_database
set contact_pref = 'ROBOCALL'
where ID = 3 AND (Phone_number IS NOT NULL AND dupe_record = 0);

# Stephan Strange
update proj2_database
set contact_pref = 'MAIL'
where id = 4 AND (Address_Line_1 IS NOT NULL AND dupe_record = 0);

# Bob's do not have phone numbers, so no update will occur
update proj2_database
set contact_pref = 'TEXT'
where id = 5 AND (Phone_number IS NOT NULL AND dupe_record = 0);

import mysql.connector
from mysql.connector import Error

def create_server_connection(host_name, user_name, user_password, db_name):
    connection = None
    try:
        connection = mysql.connector.connect(
            host=host_name,
            user=user_name,
            passwd=user_password,
            database=db_name
        )
        print("MySQL Database connection successful")
    except Error as err:
        print(f"Error: '{err}'")

    return connection

def read_query(connection, query):
    cursor = connection.cursor()
    result = None
    try:
        cursor.execute(query)
        result = cursor.fetchall()
        return result
    except Error as err:
        print(f"Error: '{err}'")
